APS Dashboard

APS Dashboard is a frontend web application built to simulate a security scan management interface. The project focuses on clean UI implementation, structured component design and rendering data from local JSON files.

* Tech Stack

React (with Vite)
React Router
Tailwind CSS
Lucide Icons

* Project Setup

Clone the repository

git clone https://github.com/Vanshika-Sharma15/aps-dashboard.git
cd aps-dashboard

Install dependencies

npm install

Run the development server

npm run dev

The application will run on:
http://localhost:5173
